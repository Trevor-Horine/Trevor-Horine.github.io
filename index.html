Junior Design Accelerated Project

In the first term of my junior design class we had a group project to build a frequency detector that would identify frequencies between middle C (261.62 Hz) and high C (523.25 Hz). My group designed an amplifier for the signal created by a microphone and used an Arduino to measure the amplified signal. An Arduino NANO was used and the analog to digital converter was configured to Serial comunication and MATLAB were used to process the signal using the built in Fast Fouier Transform (FFT) function in MATLAB. The MATLAB script then pased the frequency back to the Arduino.

Digital Logic Final Project

ECE 271 Digital Logic included a final group project that required takeing in multiple inputs in to the FPGA that we used and useing them to drive several different outputs. My part of the project took inputs from an NES controller and output to a DC motor, 7-segment dispays, and an addressable RGB LED.

Pressing the right or left sides of the dpad would spin the motor clockwise or counterclockwise. If the A button and the up button on the dpad were pressed at the same time it would incrment the blue value of the LED, pressing A and the down on the dpad would decrement the blue value. If the B button was presses in additon to the up or down on the dpad the red value would be changed, and the start button was used for the green value. The values were also output to the on board seven segment displays in two digit hexadecimal for each color in the order of red green blue from right to left.
